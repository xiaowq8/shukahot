

==**启动步骤**==

运行后台

1. 打开Redis

2. 打开数据库 

3. 打开idea

4. 打开blog_api项目

5. 运行blog_api

运行前端

1. 进入E:\git_project\blog-vue-springboot\blog-app打开终端
2. npm install （第一次运行）
3. npm run dev







# 技术

## 前端  blog-app

- Vue
- Vue-router
- Vuex
- ElementUI
- mavon-editor
- lodash
- axios
- Webpack

## 后端  blog-api

- SpringBoot
- Shiro
- Jpa
- Redis
- Fastjson
- Druid
- MySQL
- Maven

# 实现功能

## 整体

- 用户：登录 注册 退出
- 首页：文章列表、最热标签、最新文章、最热文章
- 文章分类-标签：列表、详情
- 文章归档
- 文章：写文章、文章详情
- 评论：文章添加评论 对评论回复
- 文章列表滑动分页
